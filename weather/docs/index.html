Site Plan

Site Name

This site is going to be named Better Weather. The word "better" refers to more accurate weather forecasting, easier look up of the weather, and easy to read results after a look up of the weather forecast. To go along with the site name, the domain name will be mybetterweather.com which is an available domain name.

Purpose Statement

As briefly mentioned in the site name, the purpose of this website is to make looking up the weather a better and easier experience for all who use it. The goal is to make results fast and easy to understand while still giving the most accurate results to the forecast. These goals will be accomplished by using easy to read images for each hour of the day, and not overcrowding the website so it is simple and runs quickly. To ensure accurate weather reading, the forecast will be updated often and will show a time stamp of when it was last updated. By making these extra efforts, the hope is to provide a service people will want to use above all others.
